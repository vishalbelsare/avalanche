LRSchedulerPlugin

Learning Rate Scheduler Plugin
class avalanche.training.plugins.lr_scheduling.LRSchedulerPlugin(scheduler, reset_scheduler=True, reset_lr=True)[source]¶

Bases: avalanche.training.plugins.StrategyPlugin

Learning Rate Scheduler Plugin This plugin manages learning rate scheduling inside of a strategy using the PyTorch scheduler passed to the constructor. The step() method of the scheduler is called after each training epoch.

Creates a LRSchedulerPlugin instance.

Parameters

  • scheduler – a learning rate scheduler that can be updated through a step() method and can be reset by setting last_epoch=0

  • reset_scheduler – If True, the scheduler is reset at the end of the experience. Defaults to True.

  • reset_lr – If True, the optimizer learning rate is reset to its original value. Default to True.
