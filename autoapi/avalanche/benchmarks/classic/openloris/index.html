avalanche.benchmarks.classic.openloris¶

This module contains the high-level OpenLORIS scenario/factor generator. It basically returns a iterable scenario object GenericCLScenario given a number of configuration parameters.

avalanche.benchmarks.classic.openloris.OpenLORIS(root=expanduser('~') + '/.avalanche/data/openloris/', factor='clutter')[source]¶

OpenLORIS continual scenario generator

Parameters

  • root – Path indicating where to store the dataset and related metadata. By default they will be stored in avalanche/datasets/openloris/data/.

  • factor – OpenLORIS main factors, indicating different environmental variations. It can be chosen between ‘clutter’, ‘illumination’, ‘occlusion’, ‘pixel’, or ‘mixture-iros’. The first three factors are included in the ICRA 2020 paper and the last factor (mixture-iros) is the benchmark setting for IROS 2019 Lifelong robotic vision competition.

Returns

it returns a GenericCLScenario instance that can be iterated.
